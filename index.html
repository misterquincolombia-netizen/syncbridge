2. SyncBridge Parsea Datos
Extrae cliente + pedido + lineas SKU
3. Verificar Cliente en Dolibarr
Buscar por email
EXISTS

NOT FOUND
Crear nuevo contacto
4. Crear Pedido de Venta

5. Generar Factura + Pagado

6. Log del Resultado
Exitoso o con errores parciales

Regla: Clientes

IF cliente existe por email:
  → Usar ID existente
ELSE no existe:
  → POST /thirdparties

Regla: Productos

IF SKU existe en Dolibarr:
  → Agregar linea con product_id
ELSE SKU no encontrado:
  → Log error + alerta
  → CONTINUAR con otros items
